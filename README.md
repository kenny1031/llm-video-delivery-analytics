# LLM-assisted Video Delivery Quality Analytics Platform

## LLM-assisted Experiment Readout

The system uses a local Ollama LLM to convert statistical experiment outputs into business-facing readouts.

The LLM does not make launch decisions. It summarizes SQL/statistical results generated by the experimentation layer.

## Root Cause Hypothesis Generation

The system decomposes delivery quality metrics by region, edge PoP, protocol, and IP version.  
A local Ollama LLM then summarizes the segment-level diagnostics into hypotheses for further investigation.

The LLM is not treated as the source of truth. It only generates candidate explanations based on SQL/Python diagnostics.
