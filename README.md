# LLM-assisted Creator Growth & Content Quality Analytics Platform

A data science project for creator/content analytics.

This project combines:

- SQL product metrics
- A/B testing simulation
- ML risk modeling
- LLM-assisted content enrichment
- Natural-language analytics
- Streamlit dashboard

The LLM is used as an assistant for structured content understanding, natural-language analytics, and experiment summarisation, while all decisions are grounded in SQL metrics and statistical testing.
